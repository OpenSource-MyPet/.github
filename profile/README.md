## 💻 프로젝트 소개
반려동물을 키우는 사람들을 위한 참여형 소셜 플랫폼으로, 반려인들이 개인 반려동물 포스팅, 실시간 산책 모임·채팅, 유기동물 입양 정보 조회 등 다양한 맞춤형 소통 공간을 이용할 수 있는 서비스를 제공합니다. 
또한, 가까운 동물병원 위치 제공, 이미지 기반 안질환 분석, 챗봇 상담 등 편의 기능을 통합하여 반려인의 일상을 다각도로 지원합니다. 
본 프로젝트는 이러한 기능들을 통해 반려인에게 특화된 새로운 형태의 통합 플랫폼을 구축하고, 올바른 반려문화의 확산을 목표로 하고 있습니다.  

## 👨‍💻 개발자 소개
- **김기은** : 회원 관리, 유기동물 API 기반 유기동물 보호 페이지 구현 및 상호작용(좋아요, 댓글, 북마크)기능 보조
- **손승혜** : Kakao Map API 기반 주변 동물병원 위치 정보, 지도 UI 구현 및 마이페이지 보조
- **이충희** : 반려동물 포스팅, 상호작용 기능, LLM 기반 AI 맞춤 예상 진단서, 실시간 채팅, 카카오톡 소셜로그인 및 요약 진단서 전송 구현

## 📌 프로젝트 소감
- **김기은** : 다양한 기능을 계획하고 구현하며 실제 서비스가 만들어지는 흐름을 직접 경험할 수 있었다. 팀원들과 서로 도움을 주고받으며 협업의 중요성을 다시 느낄 수 있었고, 프로젝트 전체를 하나의 결과물로 완성해가는 과정이 좋은 경험이 되었다.
- **손승혜** : 이번 프로젝트를 통해 Java와 Spring Boot를 처음 사용해 보면서 어려움도 많았지만, 직접 기능을 구현해 가는 과정 속에서 기본 구조와 동작 원리를 배우는 의미 있는 경험이 되었다. 서비스가 구성되는 흐름을 구현하며 백엔드·프론트엔드 전반의 구조를 이해할 수 있었다. 팀원들과 역할을 나누고 함께 기능을 완성해나가며 많은 경험을 쌓을 수 있었다.
- **이충희** : 팀원들과 기능을 설계하고 구현해나가는 과정 모두가 의미있었고 프로젝트에 대한 역량에 한 단계 더 나아가는 시간이 되었다 특히 일반적인 수업으로써의 프로젝트가 아닌 실제 서비스할 프로젝트로 생각하며 진행해 나가서 되게 열의를 가지고 진행할 수 있었다고 생각한다 또한 프로젝트를 주도해 나가면서 팀 단위의 의사소통 방법에 대해 더 알아가는 소중한 경험이 되었다

## 💡 향후 발전 방향
본 프로젝트는 다양한 외부 API, 이미지 처리 기능, 실시간 통신 기능 등이 결합된 서비스로, 특히 카카오 소셜API 등 일부 기능은 실제 서비스 배포 시 도메인 등록 및 사용 허가 절차가 요구되며, 해당 승인 과정이 완료되어야 정식 배포가 가능하다. 개발 과정에서 배포를 염두에 두고 준비를 진행했으나, 외부 API의 승인 절차가 학기 내에 완료되지 않아 현재는 로컬 환경에서 서비스를 구동하는 형태로 프로젝트를 마무리하였다. 향후에는 관련한 제약을 마친 뒤 서버 구축 및 서비스 안정화 과정을 거쳐 실제 운영이 가능한 형태로 확장해 나가는 것을 목표로 하고 있음
